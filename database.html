Base de datos

Las bases de datos actualmente forman parte importante de cualquier organización, ya que es indispensable guardar, organizar, acceder rápidamente y actualizar datos, es por ello que la mayoría de las empresas usan base de datos en MySQL. .

En el presente proyecto se creó una base de datos de un gimnasio con 4 tablas, en la cual una de ellas cuenta con una llave foránea para acceder a los datos de la otra tabla sin caer en redundancia y la tabla de inscripción cuenta con 2 llaves foranes ya que esta tiene que acceder a la tabla de clientes y obtener su información para realizar adecuadamente un pago y a los grupos en los cuales se inscribirán los clientes.

También he elaborado base de datos en Microsoft Access ya que es una plataforma sencilla de manejar y que gracias a su interfaz gráfica es más fácil crear una base de datos ya que no se requieren demasiados conocimientos técnico para crear una.
