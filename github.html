Repositorio de Github

También es necesario si se planea trabajar con un equipo de trabajo y mantener una buena gestión de proyectos grandes.

Otra de las funciones de Github que ocupo mucho es la de Github Pages con lo que puedo visualizar como es que quedaría una pagina web antes de subirlo a algún host de internet.

La mayoría de los proyectos que están ahí están son públicos por lo que si deseas ver como se ve te dejo el link para que lo puedas ver https://github.com/Danieru-AceRo.
