Excel

Hoy en día los datos forman parte esencial de toda organización y el programa mas popular para ello es sin duda Excel. Ya sea para organizar datos, crear gráficas, fórmulas o analizar datos tener una habilidad avanzada de usarlo es primordial.

En el curso de Platzi logré desarrollar skills como lo son:

  • Protección de datos
  • Lista de desplegables independientes
  • Base de datos
  • Separación y juntar texto
  • Fórmulas con condicionales
  • Promedio ponderado
  • Buscar H y buscar V
  • Semáforos celdas de colores
  • Tablas dinámicas
  • Gráficas dinámicas
  • Programación de macros
